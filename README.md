#  Pattern Matching Chatbot (Python)

A simple yet creative **rule-based chatbot** built using **Python, Regular Expressions, and external modules**.  
This chatbot can respond to greetings, tell jokes, solve math problems, and answer basic **Science** and **Social Studies** questions.

---

##  Features

✅ Friendly greeting with learner-focused message  
✅ Time & Date queries  
✅ Python and AI explanations  
✅ Math problem solver (e.g., `5+3`, `10/2`)  
✅ Science concepts (photosynthesis, gravity, cell, electricity, etc.)  
✅ Social Studies topics (Democracy, Indian Independence, Gandhi, Capital of India)  
✅ Jokes using the **pyjokes** module   
✅ Exit commands (`bye`, `exit`, `quit`)  

---

##  Technologies Used

- **Python 3**
- **re** (Regular Expressions)
- **datetime** (for date & time)
- **pyjokes** (for jokes)

---

##  Installation

### 1. Install Python
Make sure Python is installed:
```bash
python --version
```
```bash
pip install pyjokes
```
### 2. How to run
Save your code
```bash
chatbot.py
```
Run
```bash
python chatbot.py
```
